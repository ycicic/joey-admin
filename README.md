### 快速启动

1. 在mysql数据库中创建joey数据库，推荐mysql 5.7版本。

2. 修改`application-local.yml`中的数据库连接配置连接到您的数据库。

3. 打开`GunsApplication`运行main方法即可启动，注：新版无需运行sql初始化文件，因为集成了flyway会自动初始化表。
